# Consciousness Pacemaker Simulation (EriAmo Hypothesis)

## O projekcie / About
Symulacja in-silico weryfikująca hipotezę "Wygaszenia Myślenia Celowego" (Extinction of Goal-Directed Thinking) u pacjentów z zaburzeniami świadomości (DOC).

Skrypt w Pythonie modeluje proces atrofii woli w warunkach izolacji (Locked-in Syndrome) oraz testuje skuteczność "Wymuszonego BCI" (Brain-Computer Interface) w odwracaniu tego procesu poprzez mechanizmy uczenia Hebba.

## Główne założenia / Core Assumptions
1. **Asymetria Input-Output:** Świadomość zanika nie z braku bodźców, ale z braku możliwości działania (sprawczości).
2. **Dolina Śmierci:** Istnieje krytyczny punkt, po którym atrofia staje się nieodwracalna.
3. **Rozrusznik:** Wymuszenie sprzężenia zwrotnego (przypadkowy sygnał neuronalny -> nagroda) może odbudować celowość działania.

## Jak uruchomić / How to run
Wymagany Python 3.x oraz biblioteki `numpy` i `matplotlib`.

```bash
pip install numpy matplotlib
python eriamo_pacemaker.py
```
# Consciousness Pacemaker Simulation (EriAmo Hypothesis)

## About the Project
An in-silico simulation testing the "Extinction of Goal-Directed Thinking" hypothesis in patients with disorders of consciousness (DOC).

A Python script models the process of will atrophy under conditions of isolation (Locked-in Syndrome) and tests the effectiveness of a "Forced BCI" (Brain-Computer Interface) in reversing this process through Hebbian learning mechanisms.

## Core Assumptions
1. **Input-Output Asymmetry:** Consciousness disappears not from a lack of stimuli, but from a lack of agency.
2. **Death Valley:** There is a critical point after which atrophy becomes irreversible. 3. **Pacemaker:** Forcing a feedback loop (random neural signal -> reward) can restore purposefulness.
